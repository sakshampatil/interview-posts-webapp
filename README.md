# interview-posts-webapp

## It a simple web app where user can create or view other users posts and can view/comment on that.

## Technologies used

### Frontend

    1 Nextjs
    2 RTK
    3 RTKQ
    4 Tailwind
    5 NextUi

### Backend

    1 Node
    2 Express
    3 Bcryptjs
    4 Mongoose

### Database

    1 Mongodb

## Instructions to run the app

    1 docker-compose build
    2 docker-compose up
    3 open localhost:3000 on your browser
